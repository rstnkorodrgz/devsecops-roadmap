01

Build

Design & ship

Learn the shape of an API before the framework: resources, HTTP verbs, status codes, JSON. Then pick one stack and ship a working CRUD service with a database. For your background, Python / FastAPI is the recommended default — you already read Python, and its typed validation maps directly to secure design.

Concepts

Designing RESTful APIs — Udacity

Resources, verbs, status codes, JSON. Framework-light. Do this first — one weekend.

Free ★ Python

FastAPI courses — Class Central hub

Recommended build path. Async, Pydantic validation, DB integration, auto-generated OpenAPI docs. Filter to free.

Free Python

REST APIs with Flask & Python — Udemy

Simpler, classic hands-on build: auth, ORM, Docker, deployment via real projects.

Paid C# / .NET

ASP.NET Core Web API — Zero to Hero

The most complete free option overall: 125+ lessons → JWT, Clean Architecture, CQRS, Docker, CI/CD. Choose if you target .NET shops.

Free Node.js

RESTful API with Express — Coursera

JavaScript/TypeScript path. Common in startups; large ecosystem.

Free audit
02

Secure

Harden it

Now bolt security onto the API you just built. Add authentication (JWT / OAuth2), input validation, rate limiting, and secrets handling — then study the OWASP API Security Top 10 and map each risk back to your own code. This is where your DevSecOps background compounds.

★ Foundations

Free Standard

Free Course

Free Reference

03

Test

Prove it works

Before you attack it, learn to exercise and document the API properly — the same tools carry straight into offensive work. Postman is the industry standard for requests, collections, automated tests, and contract validation.

★ Tooling

Postman Academy

Modular, free. API exploration → collections → automated tests → CI. The tool you'll use forever, offense included.

Free Docs

OpenAPI / Swagger

Contract-first documentation. FastAPI generates this for you — learn to read, edit, and validate against the spec.

Free Practice

REST API testing courses — Class Central

Postman + curl workflows, status-code assertions, and JSON schema validation across 1,600+ free courses.

Free
04

Break

Red team

The payoff phase, and the one you're built for. Attack deliberately-vulnerable APIs (vAPI, crAPI), then turn on the service you built yourself. APIsec University is the best free resource here — and the certs are respected, culminating in a 12-hour live pentest exam.

★ Offense

API Penetration Testing (12 hrs) — APIsec University

Recon, JWT auditing, auth/BOLA attacks, mass assignment, injection, SSRF — with Postman, Burp Suite & ZAP against live labs.

Free Cert · Entry

CASA — Certified API Security Analyst

100-question, 2-hour exam (80% pass). Free, résumé-worthy quick win after the pentest course.

Free Cert · Pro

ASCP — API Security Certified Professional

The capstone: a 12-hour practical pentest of two live APIs — no multiple choice. Genuinely respected. Aim here.

Free Lab

vAPI & crAPI — vulnerable API ranges

Self-hosted intentionally-broken APIs to practice every OWASP API risk hands-on. Free and open source.

Free

Cross-cutting

Cloud provider tracks

In the real world you rarely expose an API raw — you front it with a managed gateway that handles auth, throttling, keys, and monitoring. Each major cloud has an official learning path. Layer one of these in during Phase 02–03 once your API runs locally.

Provider Service Best training Covers
AWSAmazon API Gateway Official tutorials & workshops · Educative: The Unsung Warrior REST / HTTP / WebSocket APIs, data mapping, authorizers, throttling, serverless integration.
AzureMicrosoft API Management MS Learn: Implement API Management · Explore API Management Official & free. Policies, transform/secure APIs, backend APIs, AKS integration, lifecycle.
GCPGoogle Apigee Coursera: Developing APIs with Apigee (3-course specialization) Full lifecycle across 3 courses — Design & Fundamentals, API Security (OAuth, JWT, federation), Development (traffic mgmt, portals, CI/CD).
Othersvendor-neutral Kong · Cloudflare · IBM API Connect Vendor docs & free tiers Worth a look once you know one gateway — the concepts (keys, rate limits, plugins) transfer directly.

Putting it together

A suggested 12-week run

Wk 1–2 RESTful design concepts + spin up your first FastAPI CRUD service. Phase 01
Wk 3–4 Add a database, error handling, and deploy it once (Docker + a free host). Phase 01
Wk 5–6 APIsec Fundamentals + add JWT/OAuth2, validation, rate limiting to your API. Phase 02
Wk 7 OWASP API Top 10 — audit your own code against every item. Layer in a cloud gateway. Phase 02 + Cloud
Wk 8 Postman Academy + write an automated test collection against your API. Phase 03
Wk 9–10 APIsec API Penetration Testing course against vAPI / crAPI — then your own API. Phase 04
Wk 11 Sit the CASA exam. Free, and a clean résumé line. Phase 04
Wk 12+ Work toward the ASCP practical pentest — the real capstone. Phase 04
