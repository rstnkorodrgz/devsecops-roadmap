# Contingency — CISSP (16-Week Sprint, On Demand)

> **Status: CONTINGENCY — execute only when triggered.**
> **Trigger:** ≥2 target job postings in a quarter list CISSP as **required/preferred**. Review quarterly.
> **Why demoted from the v1.x capstone:** managerial framing doesn't serve the deep-technical identity; [CCSP](../phases/phase-5-ccsp.md) provides the ISC2 credential with cloud-architecture substance instead.
> **Exam facts:** CAT format — **100–150 questions, 3 hours** (since April 2024); no skipping or revisiting questions. 5-year experience requirement + endorsement: comfortably met.

---

## The sprint (16 weeks)

With 10+ years of experience and the completed technical phases, **Domains 3, 4, 6, 7, 8 are review-only. Focus: Domains 1, 2, 5.**

| Weeks | Block |
|---|---|
| 1 | Diagnostic — full practice exam cold; map weak domains |
| 2–9 | Domain study — front-load 1, 2, 5; review-pass the rest |
| 10–13 | Practice exams — 80%+ on Boson / Destination Certification before booking confidence |
| 14–16 | Close — weakest-domain drills, "think like a risk advisor" calibration, exam |

## Focus domains

### Domain 1 — Security & Risk Management (~16%)
- [ ] Risk frameworks (NIST RMF, ISO 27005), quantification (ALE/SLE/ARO), FAIR
- [ ] Governance, compliance regimes (SOC 2, ISO 27001, GDPR + LATAM privacy), BCP vs DRP
- [ ] Policy hierarchy: policy → standard → guideline → procedure

### Domain 2 — Asset Security (~10%)
- [ ] Classification, ownership roles (owner/custodian/steward/processor/controller)
- [ ] Data lifecycle; **NIST SP 800-88** media sanitization (clear/purge/destroy)
- [ ] Privacy: minimization, purpose limitation

### Domain 5 — Identity & Access Management (~13%)
- [ ] Federation (SAML, OAuth 2.0, OIDC, SCIM), PAM/JIT, biometric metrics (FAR/FRR/CER)
- [ ] Directory services attack surface; zero-trust identity — much is review after SC-500's Entra work

## Review-only domains (one pass each)
- [ ] D3 Architecture & Engineering — crypto, security models, cloud models *(CCSP overlap)*
- [ ] D4 Network Security — *(networking foundation refreshed in Phase 0; ~13% mostly review)*
- [ ] D6 Assessment & Testing — *(daily ASPM work)*
- [ ] D7 Security Operations — IR lifecycle, forensics, DR metrics *(SC-500 Sentinel overlap)*
- [ ] D8 Software Development Security — *(Phase 3 + SSDF, strongest domain)*

## Exam & after
- [ ] Two full-length practice exams at 80%+ → sit it → ✅ **PASS CISSP**
- [ ] Endorsement (5 yrs experience), AMF; CPEs pool with CCSP under ISC2
