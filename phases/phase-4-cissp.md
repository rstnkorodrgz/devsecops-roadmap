## 🗓️ Weeks 1–2 — CISSP Domains 1–4

### Domain 1 — Security & Risk Management (15%)
- [ ] Understand CIA triad, security governance, and organizational roles (CISO, DPO, etc.)
- [ ] Study risk management frameworks — NIST RMF, ISO 27005
- [ ] Learn risk quantification — ALE, SLE, ARO, AV calculations
- [ ] Understand FAIR model — factor analysis of information risk
- [ ] Study compliance frameworks — SOC 2, ISO 27001, NIST CSF, HIPAA, PCI-DSS
- [ ] Learn threat intelligence concepts — tactical, operational, strategic
- [ ] Understand business continuity vs disaster recovery (BCP vs DRP)
- [ ] Study security policies hierarchy: policy → standard → guideline → procedure

### Domain 2 — Asset Security (10%)
- [ ] Understand data classification — public, internal, confidential, restricted
- [ ] Learn data ownership roles — owner, custodian, steward, processor, controller
- [ ] Study data lifecycle — creation, storage, use, sharing, archival, destruction
- [ ] Understand data remanence and secure disposal methods (DoD 5220.22-M, degaussing)
- [ ] Learn privacy concepts — GDPR, CCPA, data minimization, purpose limitation

### Domain 3 — Security Architecture & Engineering (13%)
- [ ] Study security models — Bell-LaPadula, Biba, Clark-Wilson
- [ ] Understand evaluation criteria — Common Criteria, FIPS 140-2/3
- [ ] Learn cryptography fundamentals — symmetric, asymmetric, hashing, PKI
- [ ] Study secure design principles — defense in depth, least privilege, fail-safe defaults
- [ ] Understand virtualization security, cloud security models (IaaS/PaaS/SaaS responsibility)
- [ ] Learn industrial control systems (ICS/SCADA) security considerations

### Domain 4 — Communication & Network Security (13%) ← Your CCNA advantage
- [ ] Review OSI and TCP/IP models at conceptual security level
- [ ] Study network protocols with security lens — DNS security (DNSSEC), HTTPS, SSH
- [ ] Understand network segmentation — DMZ, VLANs, microsegmentation
- [ ] Learn VPN types — IPSec, SSL/TLS, L2TP — and their security tradeoffs
- [ ] Study wireless security — WPA3, EAP methods, rogue AP detection
- [ ] Understand SD-WAN security implications
- [ ] Review firewall types — packet filter, stateful, NGFW, WAF
- [ ] Study zero-trust network architecture (ZTNA) principles

## 🗓️ Weeks 3–4 — CISSP Domains 5–8

### Domain 5 — Identity & Access Management (13%)
- [ ] Understand authentication factors and MFA
- [ ] Study federated identity — SAML, OAuth 2.0, OIDC, SCIM
- [ ] Learn PAM (Privileged Access Management) — just-in-time access, session recording
- [ ] Understand Zero Trust identity — continuous verification, device trust
- [ ] Study biometric authentication — accuracy metrics (FAR, FRR, CER)
- [ ] Learn directory services security — LDAP, Active Directory attack surface

### Domain 6 — Security Assessment & Testing (12%)
- [ ] Understand vulnerability assessment vs penetration testing vs red teaming
- [ ] Study penetration testing phases — reconnaissance, scanning, exploitation, post-exploitation, reporting
- [ ] Learn log review and monitoring — SIEM concepts, correlation rules
- [ ] Understand code review methodologies — static vs dynamic analysis
- [ ] Study audit types — internal, external, third-party (SOC 2 Type I vs II)
- [ ] Learn synthetic transactions and real user monitoring concepts

### Domain 7 — Security Operations (13%) ← Security+ coverage
- [ ] Understand incident response lifecycle — NIST SP 800-61
- [ ] Study digital forensics — chain of custody, order of volatility, disk forensics
- [ ] Learn threat hunting concepts — hypothesis-driven, IoC-driven
- [ ] Understand change management and configuration management
- [ ] Study disaster recovery — RTO, RPO, MTTR, MTBF
- [ ] Learn physical security concepts — mantrap, CCTV, access control systems

### Domain 8 — Software Development Security (11%)
- [ ] Understand SDLC security integration — waterfall vs agile security approaches
- [ ] Study secure coding standards — OWASP, CERT
- [ ] Learn DevSecOps maturity models — BSIMM, OWASP SAMM
- [ ] Understand database security — SQL injection prevention, stored procedures
- [ ] Study software supply chain security — SBOMs, dependency management
- [ ] Review your Phase 2–3 knowledge through the CISSP lens

## 🗓️ Weeks 7–8 — CISSP Exam Prep & Final Review

### Study Strategy
- [ ] Take full-length CISSP practice exam (250 questions) — Session 1 — record score
- [ ] Review all incorrect answers — map each to a domain and re-study
- [ ] Take full-length CISSP practice exam — Session 2 — record score
- [ ] Review CISSP "think like a manager" decision framework
- [ ] Revisit all weakest domains (those below 70% in practice exams)
- [ ] Read all CISSP exam tips from ISC2 official site

### Exam Prep Materials
- [ ] Complete (ISC)² Official CISSP Study Guide — all chapters
- [ ] Complete Thor Pedersen's CISSP course (Udemy) — watch all videos
- [ ] Complete Destination Certification MindMap series (YouTube — free)
- [ ] Score 80%+ on Boson or Destination Certification practice exams
- [ ] Schedule CISSP exam at Pearson VUE

### Exam Day
- [ ] Bring two forms of valid ID
- [ ] Arrive 30 minutes early
- [ ] Remember: CISSP is adaptive (CAT format) — 125–175 questions, 4 hours
- [ ] Answer as a "senior security manager" not a technician
- [ ] ✅ **PASS CISSP — Certified Information Systems Security Professional**

### Post-CISSP
- [ ] Submit CISSP endorsement application (requires 5 years experience)
- [ ] Identify a current CISSP member to endorse your application
- [ ] Pay ISC2 annual maintenance fee and AMF
- [ ] Plan CPE (Continuing Professional Education) strategy — 120 CPEs over 3 years
